CouchCMS Configuration Settings
================================

After uploading couch/ folder, edit couch/config.php with these settings:

// MySQL Database Settings (Get from CrazyDomains cPanel)
define( 'K_DB_NAME', 'your_database_name' );
define( 'K_DB_USER', 'your_database_user' );
define( 'K_DB_PASSWORD', 'your_database_password' );
define( 'K_DB_HOST', 'localhost' ); // Usually 'localhost' for CrazyDomains

// Site URL
define( 'K_SITE_URL', 'https://mindfuljourneypsychology.com.au/' );

// Admin folder (default is 'couch')
define( 'K_ADMIN_URL', K_SITE_URL . 'couch/' );

// Other settings can stay default

Admin panel will be at: https://mindfuljourneypsychology.com.au/couch/
